### 概要
このように，ブログにスライドをいい感じに埋め込めるようになりました。

<Slide src="https://speakerdeck.com/player/0bf3f3abf9044ff9be9f752d8028ff0d?slide=18"/>

### 方法

埋め込み用の HTML は，以下の API から取得できます。

```bash
https://speakerdeck.com/oembed.json\?url={slide_url}
```

`{slide_url}` には，Speakerdeck のスライドページの URL を指定してください。

これをブラウザに入力するなり `curl` で GET すると，json が帰ってきます。その json に [こんな感じ 🔍](https://speakerdeck.com/oembed.json/?url=https://speakerdeck.com/kenmikanmi/emarugorizumu-from-machine-learning-a-probabilistic-perspective)
で，`html` というキーがあるのがわかると思います。これが埋め込み用の HTML です。

構造は以下のような感じになってます。

```html
<div style="{div_styles}">
  <iframe src="//speakerdeck.com/player/{slide_id}"
    style="{iframe_styles}">
  </iframe>
</div>
```

注：`{slide_id}` は `{slide_url}` とは異なるものです。


参考までにこのブログでは，style を以下のようにしてます。

```html
<div style="
  position: relative;
  display:block;
  margin:0 auto;
  width: 100%;
  max-width:780px;
  max-height: 585px;
  padding-bottom: 75%;
  top: 50%;"
>
  <iframe 
    src="//speakerdeck.com/player/0bf3f3abf9044ff9be9f752d8028ff0d?slide=18"
    style="
      position: absolute;
      top: 0;
      left: 0%;
      width: 100%;
      height: 100%;
      max-width:780px;
      max-height: 585px;
      border: 0;
    "
  >
  </iframe>
</div>
```

### React コンポーネントにして，MDX に埋め込む

ここから先は，このブログ特有の話です。

このブログでは，マークアップに MDX というのを使っています。MDX というのは *JSX embedded in Markdown*，
はしょっていうと，React コンポーネントをマークダウンに埋め込んで使えるフォーマット形式です。

さっきの HTML のあれをコンポーネント化すると，以下のような感じで Markdown に埋め込めるようになります。

例）
```jsx
<Slide src="https://speakerdeck.com/player/{slide_id}"/>
```

定義したコンポーネントは以下です。

```jsx
import React from 'react';
import styled from '@emotion/styled';

const IFrame = styled.iframe`
  position: absolute;
  top: 0;
  left: 0%;
  width: 100%;
  height: 100%;
  max-width:780px;
  max-height: 585px;
  border: 0;
`

const IFrameWrapper = styled.div`
  position: relative;
  display:block;
  margin:0 auto;
  width: 100%;
  max-width:780px;
  max-height: 585px;
  padding-bottom: 75%;
  top: 50%;
  // transform:translateY(-50%)
`

const Slide = ( props ) => {
  return (
    <IFrameWrapper>
        <IFrame src={ props.src } allow={"fullscreen"}>
        </IFrame>
    </IFrameWrapper>
  );
};

export default Slide;

```
